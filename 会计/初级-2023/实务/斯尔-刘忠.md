# 实务-刘忠

## 快速入门

1. 基本职能
   1. 核算: 确认、计量、记录、报告
   2. 监督: 真实性、合法性、合理性
2. 拓展职能
   1. 预测经济前景
   2. 参与经济决策
   3. 评价经营业绩
3. 基本假设
   1. 会计主体
   2. 持续经营
   3. 会计分期
   4. 货币计量
4. 会计基础
   1. 权责发生制
   2. 收付实现制
5. 会计要素
   |            | 定义                                 | 确认                                     |
   | :--------: | :----------------------------------- | :--------------------------------------- |
   |    资产    | 过去, 拥有或控制, 经济利益流入       | 很可能流入, 可靠计量                     |
   |    负债    | 过去, 经济利益流出, 现时义务         | 很可能流出, 可靠计量                     |
   | 所有者权益 | 股东权益, 剩余权益                   | 资产 - 负债                              |
   |    收入    | 日常活动和与之相关的活动             | 控制权转移                               |
   |    费用    | 日常活动                             | 很可能流出, 可靠计量, 资产减少或负债增加 |
   |    利润    | 利润总额 = 收入 - 费用 + 利得 - 损失 |                                          |
6. 要素计量
    |            | 定义                       | 使用                                           |
    | :--------: | :------------------------- | :--------------------------------------------- |
    |  历史成本  | 实际成本                   | 大部分资产和负债的计量                         |
    |  重置成本  | 现行成本                   | 盘盈固定资产                                   |
    | 可变现净值 |                            | 期末存货的计量                                 |
    |    现值    | 未来的终值折现到当下的价值 | 租赁负债的确认                                 |
    |  公允价值  | 市场参与者均认可的价值     | 交易性金融资产, 采用公允价值计量的投资性房地产 |
7. 会计等式
   1. 资产 = 负债 + 所有者权益
   2. 资产 - 负债 = 所有者权益
   3. 收入 - 费用 = 利润
8. 复式记账法
   1. 帐户: 带有结构的科目, 丁字结构
   2. 科目: 约等于帐户, 主要使用对象
9. 科目分类
    |        类别        | 主要科目                                                                                   |
    | :----------------: | :----------------------------------------------------------------------------------------- |
    |       资产类       | 库存现金、银行存款、其他货币资金、应收帐款、原材料、库存商品、固定资产、在建工程、无形资产 |
    |       负债类       | 应缴税费、应付帐款、应付职工薪酬、长期借款                                                 |
    |    所有者权益类    | 实收资本、资本公积                                                                         |
    |    成本类(资产)    | 生产成本(直接费用)、制造费用(间接费用)、研发支出                                           |
    | 损益类(收入、支出) | 主营业务收入、其他业务收入、主营业务成本、其他业务成本、管理费用、销售费用、财务费用       |
    |       共同类       |                                                                                            |

## 第一章 概述

1. 会计
   1. 特征
      1. 以货币为主要计量单位
      2. 准确性、完整性、连续系统性
   2. 职能
      1. 基本
         1. 核算: 以货币为主要计量单位, 对特定经济活动主体确认、计量、记录、报告; 与资金相关
         2. 监督: 真实性、完整性、合法性、合理性; 审核、审查
      2. 拓展
         1. 预测经济前景
         2. 参与经济决策
         3. 评价经营业绩
   3. 目标
      1. 反映企业管理层受托责任履行情况
      2. 有助于财务报告使用者作出经济决策
2. 基本假设
   1. 会计主体: 空间范围; 会计主体未必是法律主体, 企业集团不是法律主体、是会计主体
   2. 持续经营: 不会停业
   3. 会计分期: 以货币为主要计量单位
   4. 货币计量: 划分为一个连续的、长短相同的期间, 少于一个完整年度的为中期
   5. 基础
      1. 权责发生制: 所有企业、政府财务会计
      2. 收付实现制: 政府预算会计
3. 信息质量要求
   1. 可靠性: 实际发生
   2. 相关性: 经济决策需求相关
   3. 可理解性: 清晰明了
   4. 可比性
      1. 同一企业不同期间可比
      2. 不同企业同一期间可比
      3. 企业可以变更会计信息、但不得随意变更
   5. 实质重于形式: 经济实质重于法律形式
   6. 重要性: 功能、性质、金额
   7. 谨慎性: 不应高估资产或收益, 不应低估负债或费用, 计提
   8. 及时性: 不得提前或延后
4. 职业道德
   1. 诚实守信
   2. 客观公正
   3. 参与管理
5. 内控
   1. 对象: 全体员工、董事会、监事会、经理层
   2. 过程
      1. 企业生产经营管理活动的全过程
      2. 企业风险控制的全过程
      3. 信息收集、整理、传递与运用的全过程
   3. 作用
      1. 提高会计信息质量
      2. 合理保证企业合法合规
   4. 目标
      1. 合理保证企业经营管理的合法合规
      2. 资产安全完整
      3. 财务报告及相关信息的真实完整
      4. 提高经营效率和效果
      5. 促进企业实现战略发展
   5. 要素
      1. 内部环境: 五要素之道
      2. 风险评估: 重要环节
      3. 控制活动
      4. 信息与沟通
      5. 内部监督: 重要保证

## 第二章

1. 资产
   1. 定义/特征: 过去的, 企业拥有或控制, 预期带来经济利益
   2. 确认
      1. 经济利益很可能流入企业
      2. 成本或价值能够可靠地计量
2. 负债
   1. 定义/特征: 过去的, 导致经济利益流出企业的现时义务
   2. 确认
      1. 经济利益很可能流出企业
      2. 金额能够可靠地计量
3. 所有者权益
   1. 定义: 股东权益, 亦称净资产
