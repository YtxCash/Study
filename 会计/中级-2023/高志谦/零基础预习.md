# 高志谦-会计实务-零基础预习

## 专题一

### 第一讲

1. 财务会计: 原始凭证 - 记账凭证 - 帐簿 - 报表, 对外服务, 规范化, 相当于把企业的资金运动翻译成会计语言
2. 管理会计: 对内, 灵活性
3. 财报目标
   1. 向财报使用者提供决策有用信息, 相关性, 一般的、中性的满足, 要不偏不倚
   2. 反映管理层受拖责任履行情况
4. 财报使用者
   1. 外人
      1. 投资人
      2. 债权人
      3. 政府
      4. 持币观望者
   2. 内人
      1. 经营者
5. 股份制有限责任公司
   1. 同股同权
   2. 财产所有权和经营权是分离的
   3. 架构
      1. 股东大会
      2. 董事会
      3. 董事长
         1. CEO
         2. CFO
         3. COO
         4. 董秘

### 第二讲

1. 会计主体
   1. 空间范围, 能独立核算
   2. 法人一定是会计主体, 会计主体不一定是法人
   3. 法人相对独立, 不能套娃
2. 持续经营
   1. 假定企业不死, 时间范围
3. 会计分期
   1. 人为分期
4. 货币计量
   1. 无差别性、统一性

### 第三讲

1. 权责发生制
   1. 更加严谨, 是关键的、核心的基础, 应计制
2. 收付实现制
   1. 根据重要性原则, 不重要的信息简化使用收付实现制
3. 信息质量要求
   1. 可靠性: 真实
   2. 相关性: 有用
   3. 可理解性: 信息完整、清晰明了
   4. 可比性: 信息口径要保持一致、不得随意变更; 政策变更要尽可能追溯
   5. 实质重于形式: 经济实质、法律形式
   6. 重要性: 抓重点
      1. 金额达到一定程度
      2. 指标本质就很重要
   7. 谨慎性
      1. 资产减值
      2. 加速折旧
   8. 及时性: 就是要快

### 第四讲

1. 资产
   1. 过去的, 很可能(> 50%)生钱的资源
   2. 拥有或控制
2. 负债
   1. 过去的, 很可能(> 50%)花钱的现时义务
3. 所有者权益
   1. 剩余权益
4. 利润 = 收入 - 费用 + 资产处置损益 + 营业外收入 - 营业外支出
5. 营业利润: 收入 - 费用 + 资产处置损益
6. 直接计入当期损益的利得和损失: 资产处置损益 +营业外收入 - 营业外支出
7. 会计对象 - 资金运动 -(分类) 会计要素 -(再分类) 会计科目
8. 静态反映财务状况: 资产 = 负债 + 所有者权益
9. 动态反映经营成果: 收入 - 费用 = 利润

### 第五讲

1. 流动资产
   1. 货币资金
      1. 银行存款
      2. 库存现金
      3. 其他货币资金
         1. 银行汇票: 异地结算
         2. 银行本票: 本地结算
         3. 信用卡存款
         4. 信用证保证金存款: 国际结算、银行担保
         5. 存出投资款
         6. 外埠存款
   2. 应收票据
      1. 本质是借条
      2. 商业承兑汇票: 信用度非常低、几乎不用
      3. 银行承兑汇票: 信用度高很多, 市面上流通的

### 第六讲

1. 非流动资产
   1. 股票: 股东身份证
      1. 资产(银行存款)增加, 所有者权益(股本, 资本公积-股本溢价)增加
      2. 长期股权投资: 对被投资公司要达到重大影响才算
         1. 控制: > 50%
         2. 共同控制
         3. 重大影响: > 20%
      3. 其他权益工具投资
      4. 交易性金融资产
   2. 债权
      1. 资产(银行存款)增加, 负债(应付债券)增加
      2. 债权投资
      3. 交易性金融资产(短线)
      4. 其他债权投资
   3. 固定资产: 价值高、不易损耗, 3 科目、5 指标
      1. 固定资产: 只登原价、初始成本
      2. 累计折旧
      3. 固定资产减值准备
   4. 无形资产: 商标权、专利权、专有技术、版权、特许经营权、土地使用权
      1. 无形资产
      2. 累计摊销
      3. 无形资产减值准备

## 专题二

1. 增值税
   1. 价外税, 不影响损益
   2. 无所不征, 天下第一税
   3. 关税 = 买价 \* 关税率
   4. 进口货物进项税 = (买价 + 关税) \* 税率
2. 消费税
   1. 离开能活的不征消费税
   2. 影响损益, 借: 税金及附加, 贷: 应缴税费-消费税
3. 复利折现系数
4. 年金折现系数
