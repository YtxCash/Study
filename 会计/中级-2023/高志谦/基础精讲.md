# 2023-中级

## 第一章 概述

1. 会计: 描述反映资金运动, 是一种信息加工系统; 原始凭证 - 记账凭证 - 帐簿 - 财报
2. 财报使用人
   1. 外人
      1. 投资人: 保值、增值
      2. 债权人: 偿债能力, 现金流量表
      3. 政府部门: 是否足额纳税, 稽查、税收
      4. 持币观望者
   2. 内人
      1. 经营管理者
3. 财报: 要求不偏不倚, 中性满足
4. 母子公司
   1. 持股过半
   2. 完成控制
5. 法人之间是相互独立的, 绝对不存在套娃
6. 基本假设
   1. 会计主体: 个人行为不得和企业行为混在一起
   2. 持续经营: 企业不死
   3. 会计分期
      1. 分段提供信息, 实时反映和监督
   4. 货币计量
      1. 会计仅反映那些能以货币表达的信息
      2. 币值稳定假设
7. 信息质量要求
   1. 可靠性: 真实; 真实可靠、内容完整
   2. 相关性: 有用, 中性满足、不偏不倚
   3. 可理解性: 看得懂, 清晰明了
   4. 可比性
      1. 纵向可比: 选择政策后、轻易不要动
      2. 横向可比
      3. 处置股份导致影响力降级, 从控制转为重大影响, 剩余股份由原成本法核算追溯调整至权益法核算
      4. 会计政策变更而进行的追溯调整法
      5. 将子公司会计政策、会计期间调整至母公司标准的合并报表准备工作
   5. 实质重于形式
      1. 合并报表的编制
      2. 售后回购: 抵押借入, 到期后还款、还利息
      3. 分期收款销售商品
         1. 现销价实现收入
         2. 收入转为借出, 实现利息
   6. 重要性
      1. 金额规模到达一定程度
      2. 指标本质上属于重要信息, 比如净利润
      3. 合并报表中抵消内容的选择
      4. 中报没年报那么详细
      5. 存货盘盈和固定资产盘盈的差异处理
   7. 谨慎性
      1. 资产减值准备的计提
      2. 加速折旧法
      3. 或有事项的处理
   8. 及时性
8. 要素
   1. 资金运动 -(分类) 会计要素 -(再分类) 会计科目
   2. 财务状况三要素、静态
      1. 资产 = 负债 + 所有者权益
   3. 经营成果三要素、动态
      1. 收入 - 费用 = 利润
   4. 资产
      1. 过去的事实
      2. 企业拥有或控制
      3. 很可能生钱的资源(>50%)
      4. 确认
         1. 相关的经济利益很可能流入企业, >50%
         2. 金额能够可靠地计量
   5. 负债
      1. 过去的事实
      2. 很可能导致经济利益流出, >50%
      3. 现时义务, 法律义务
      4. 确认
         1. 很可能导致经济利益流出, >50%
         2. 金额能够可靠地计量
   6. 所有者权益
      1. 流入资本
         1. 股本(实收资本)
         2. 资本公积-股本溢价(资本溢价)
      2. 留存收益
         1. 盈余公积
         2. 利润分配-未分配利润
      3. 其他综合收益: 直接记入所有者权益的利得和损失
         1. 利得: 非日常
            1. 影响损益
               1. 贷: 资产处置损益
                  1. 出售固定资产
                  2. 出售无形资产
               2. 贷: 营业外收入
                  1. 罚款
                  2. 无法偿付的应付帐款
                  3. 现金盘盈
                  4. 权益法下初始投资成本小于所占净资产份额的差额
            2. 直接计入所有者权益, 注意处置时账务处理
               1. 其他权益工具投资的暂时增值, 处置时转留存收益
               2. 权益法下被投资方的其他权益工具投资增值造成的投资增值, 处置时转留存收益
               3. 其他债权投资的增值, 处置时转投资收益
               4. 权益法下被投资方其他综合收益增值造成的投资增值, 处置时转投资收益
               5. 债权投资重分类为其他债权投资形成的增值, 处置时转投资收益
               6. 自用房地产、存货转为公允模式的投资性房地产形成的增值, 处置时转其他业务成本
               7. 重新计量设定受益计划净资产、净负债形成的价值差额, 贷记其他综合收益, 将来不得转入损益
               8. 权益法下被投资方重新计量设定受益计划净资产、净负债形成的其他综合收益造成的投资增值, 贷记其他综合收益, 将来不得转入损益
         2. 损失: 非日常
            1. 影响损益
               1. 借: 资产处置损益
                  1. 出售固定资产
                  2. 出售无形资产
               2. 借: 营业外支出
                  1. 罚没支出
                  2. 违约金
                  3. 盘盈固定资产
                  4. 盘亏存货
            2. 直接计入所有者权益
               1. 自用房地产、存货转为公允模式的投资性房地产形成的减值, 记入公允价值变动损益, 不属于所有者权益
               2. 其他和利得反过来
   7. 收入: 日常
      1. 合同各方已批准该合同并承诺将履行各自义务
      2. 合同明确了合同各方与所转让商品或提供劳务相关的权利或义务
      3. 该合同有明确的与所转让商品或提供劳务相关的支付条款
      4. 合同具有商业性质: 双方交换的东西不同质
      5. 企业因向客户转让商品或提供劳务而有权取得的对价很可能收回, 可能性>50%
   8. 费用: 日常
      1. 经济利益很可能流出, >50%
      2. 导致资产减少或负债增加
      3. 利益流出额能够可靠地计量
   9. 利润 = 收入 - 费用 + 资产处置损益 + 营业外收入 - 营业外支出
      1. 营业利润主体 = 收入 - 费用
      2. 营业利润整体 = 收入 - 费用 + 资产处置损益
      3. 直接计入当期损益的利得和损失 = 资产处置损益 + 营业外收入 - 营业外支出
